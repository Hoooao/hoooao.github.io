1.1 排序

    快速排序：

int arr[N];

void quick_sort(int arr[],int l,int r){
    if(l>=r) return;
    //从左右端点两侧开始，因为下面do/while
    int i=l-1,j=r+1;
    //x取中间值可以防止边接问题
    int x = arr[l + r >> 1];
    while(i<j){
        do(i++);while(arr[i]<x);
        do(j--);while(arr[j]>x);
        //交换
        if (i < j) swap(arr[i], arr[j]);
    }
    quick_sort(arr,l,j);
    quick_sort(arr,j+1,r);
}

    归并排序

void merge_sort(int q[], int l, int r)
{
    if (l >= r) return;
    //直接取中间值
    int mid = l + r >> 1;

    merge_sort(q, l, mid), merge_sort(q, mid + 1, r);

    int k = 0, i = l, j = mid + 1;
    while (i <= mid && j <= r)
        if (q[i] <= q[j]) tmp[k ++ ] = q[i ++ ];
        else tmp[k ++ ] = q[j ++ ];
    //把剩余的假如数组
    while (i <= mid) tmp[k ++ ] = q[i ++ ];
    while (j <= r) tmp[k ++ ] = q[j ++ ];
    //放回原数组
    for (i = l, j = 0; i <= r; i ++, j ++ ) q[i] = tmp[j];
}






1.2 二分

    最经典的二分就是二分查找，所有二分题都有类似的思路：想出一个check函数去检查当前二分点位是否满足某条性质（在二分查找中就是mid小于大于等于目标值），然后对不满足的情况进行if/else处理。虽然二分查找中依靠整个书里的单调性，但是二分本身是不需要单调性的：最重要的问题是边界（满足/不满足某个性质的边界）的判断。而二分的目的常常也是找到条件的边界。

1.2。1 整数二分

    首先假设红色区域代表符合某一性质的部分，绿色区域代表不符合，希望找到红色的边界。使用mid=(l+r+1)/2得到二分，然后使用check(mid)检查当前位置是否符合该性质：若符合 （落在了红色）那么更新下一次二分的范围为[mid,r]（需要包含mid，因为mid本身也可能是边界）；若不符合（落在了绿色），更新二分范围为[l,mid-1]（不包含mid因为mid必然不是符合某性质的边界）。

    另一种情况下，假设红色区域代表不符合某一性质的部分，绿色区域代表符合，希望找到绿色的边界。使用mid=(l+r)/2得到二分，然后使用check(mid)检查当前位置是否符合该性质：若符合 （落在了绿色）那么更新下一次二分的范围为[l,mid]（需要包含mid，因为mid本身也可能是边界）；若不符合（落在了红色），更新二分范围为[mid+1,r]（不包含mid因为mid必然不是符合某性质的边界）。

    可以看到如果我们想要找的边界是在形式上的左侧还是右侧会影响二分公式本身 以及更新后的二分范围。二分范围的变化很好理解，但问题是什么时候使用mid=(l+r+1)/2，什么时候使用mid=(l+r)/2。

    在C++中整形自动向下取整，假如在第一种情况中没有+1：若l=r-1，也就是说再来一轮操作就能确定边界在哪/是否存在了，mid=(l+r)/2会导致mid=1。若check(mid)为真，二分会进入[mid,r]，也就是[l,r]，即死循环。但是在第二种情况中不会出现这种情况：因为当l=r-1时，若check(mid)为真则变换范围为[l,l]，反之r,r，无论如何都结束循环且得到答案。

    这种情况是固定的，所以可以记住，只要范围变换的规律符合两种情况之一，那么对应的mid的计算公式就直接使用即可。

    例题：给定一个按照升序排列的长度为 n 的整数数组，以及 q 个查询。对于每个查询，返回一个元素 k 的起始位置和终止位置（位置从 0 开始计数）。如果数组中不存在该元素，则返回 -1 -1。

输入格式：

  第一行包含整数 n 和 q，表示数组长度和询问个数。第二行包含 n 个整数（均在 1∼10000 范围内），表示完整数组。接下来 q 行，每行包含一个整数 k，表示一个询问元素。

输出格式：

  共 q 行，每行包含两个整数，表示所求元素的起始位置和终止位置。如果数组中不存在该元素，则返回 -1 -1。

int n,m,p[N];

int main()
{
    cin>>n>>m;
    for (int i = 0; i < n; i ++ ) scanf("%d", &p[i]);
    
    while (m -- ){
        int x;
        scanf("%d", &x);
        int l=0,r=n-1;
        while(l<r){
            int mid=l+r>>1;
            if(p[mid]>=x) r=mid;
            else l=mid+1;
        }
        //没找到
        if(p[l]!=x) printf("-1 -1\n");
        else{
            printf("%d",l);
            l=0,r=n-1;
            while(l<r){
                int mid=l+r+1>>1;
                if(p[mid]<=x) l=mid;
                else r=mid-1;
            }
            printf(" %d\n",l);
        }
    }
}

首先画一个图，随便找两个点当x和mid，由于先输出起始点，就先思考“不符合的在左侧，符合的在右侧的情况”，然后得到范围变换的方式，便确定了mid的公式。之后输出终止点，也是一样。

1.2.2 浮点数二分

    相比于整数二分，浮点数二分不需要关心边界问题，因为不会取整。l与r可能永远不会相等（double），所以只要当l-r小于某个很小的值的时候，就说它俩相等

    例题：开平方根

int main(){
    double x;
    cin>>x;
    double l=0,r=x;
    double mid;
    //这里的差值最好比题目中精度高2
    while(r-l>1e-6){
        mid=(l+r)/2;
        if(mid*mid>x) r=mid;
        else l=mid;
    }
    cout<<mid;
}
